\subsubsection{\setlength{\rightskip}{0pt plus 5cm}{\bf GtvCell}$\ast$ gtv\_\-point\_\-locate (GtsPoint $\ast$ {\em p}, {\bf GtvVolume} $\ast$ {\em volume}, {\bf GtvCell} $\ast$ {\em guess})}\label{group__locate_gdf51c0d369beced1cd73c1196f0ad08a}


Locate a point in a triangulation using a remembering stochastic walk (Olivier Devillers, Sylvain Pion and Monique Teillaud, 2002, {\tt `Walking in a triangulation'}, International Journal of Foundations in Computer Science, 13:181--199). Note that at present this will not work for volumes with holes in their interior.

\begin{Desc}
\item[Parameters:]
\begin{description}
\item[{\em p}]a GtsPoint to locate; \item[{\em volume}]a \doxyref{GtvVolume}{p.}{structGtvVolume}; \item[{\em guess}]a \doxyref{GtvCell}{p.}{structGtvCell} which is an initial guess for a cell containing {\em p\/} or NULL.\end{description}
\end{Desc}
\begin{Desc}
\item[Returns:]a \doxyref{GtvCell}{p.}{structGtvCell} containing {\em p\/} or NULL if {\em p\/} is not in {\em volume\/}. \end{Desc}

Find a \doxyref{GtvCell}{p.}{structGtvCell} in a given volume which encloses a GtsPoint by sequentially testing the cells. This is not a very good way of locating a point but it does work and can be used in testing other point location methods.

\begin{Desc}

\item[{\em p}]a GtsPoint; \item[{\em volume}]a \doxyref{GtvVolume}{p.}{structGtvVolume} to search for the location of {\em p\/}; \item[{\em guess}]ignored, included for compatibility with other location functions.\end{description}
\end{Desc}
\begin{Desc}
\item[Returns:]a \doxyref{GtvCell}{p.}{structGtvCell} containing {\em p\/} or NULL if {\em p\/} is not in {\em v\/}. \end{Desc}
